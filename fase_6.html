Nuestra APi

Nuestra API de suplementos deportivos, disponible en este enlace, https://api-suplementos.usbtopia.usbbog.edu.co/suplementos proporciona un catálogo completo de información detallada sobre diversos suplementos. Esta API ha sido diseñada para facilitar el acceso a datos como el nombre del suplemento, su categoría, tabla nutricional, ingredientes, y el modo de uso recomendado. Gracias a su arquitectura RESTful, permite a los desarrolladores realizar consultas mediante solicitudes HTTP para obtener listados de suplementos, buscar suplementos específicos por nombre o categoría, y acceder a descripciones detalladas de cada producto. Esta API es ideal para integrarla en aplicaciones web y móviles que requieran un recurso confiable para la búsqueda y visualización de información sobre suplementos deportivos, optimizando la experiencia del usuario con datos precisos y actualizados.

Nuestra aplicacion móvil

Nuestra aplicación móvil, desarrollada con React Native y Expo Go, ofrece una experiencia fluida e intuitiva para los usuarios interesados en suplementos deportivos. Utilizando JavaScript, hemos creado una plataforma versátil que permite a los usuarios explorar un extenso catálogo de suplementos. La aplicación incluye funcionalidades como la búsqueda de productos por nombre o categoría, la visualización de detalles como la tabla nutricional, ingredientes y modo de uso. Gracias a la integración con nuestra API, los datos se actualizan en tiempo real, brindando a los usuarios información precisa y relevante. Esta combinación de tecnologías nos ha permitido desarrollar una solución multiplataforma eficiente, garantizando una excelente experiencia tanto en dispositivos iOS como Android.
